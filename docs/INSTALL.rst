wcc.carousel Installation
-------------------------

To install wcc.carousel using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.carousel`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.carousel

* Re-run buildout, e.g. with:

    $ ./bin/buildout
